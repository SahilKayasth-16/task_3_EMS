# Employee Management System (MERN)

A full-stack Employee Management System built using:

- React.js + Vite
- Node.js
- Express.js
- MongoDB
- Axios

## Features

- Add Employee
- Edit Employee
- Delete Employee
- View Employees
- Form Validation
- Responsive Design
- REST API Integration

## Tech Stack

Frontend:
- React
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

## Run Project

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
